# Таксопарк - Диспетчерская система

## Описание проекта

Проект представляет собой комплексную систему для управления таксопарком, состоящую из двух приложений:

1. **Диспетчерская (ПК)** - приложение на Python для управления заказами и водителями
2. **Водительский терминал (Android)** - мобильное приложение для просмотра новых заказов

Оба приложения работают с единой базой данных MySQL, размещенной на ПК через XAMPP.

---

## Установка и настройка

### 1. Установка XAMPP

1. Скачайте XAMPP с официального сайта: https://www.apachefriends.org/
2. Запустите установщик и установите XAMPP в папку `C:\xampp`
3. После установки запустите **XAMPP Control Panel**
4. Нажмите кнопки **Start** напротив:
   - **Apache** (веб-сервер)
   - **MySQL** (база данных)

### 2. Создание базы данных

База данных предоставлена отдельным файлом `taxi_dispatch.sql`. Для ее создания:

1. Откройте браузер и перейдите по адресу: `http://localhost/phpmyadmin`
2. Нажмите на вкладку **Import** (Импорт)
3. Нажмите **Выберите файл** и укажите файл `taxi_dispatch.sql`
4. Нажмите **Выполнить**

### 3. Установка PHP API для мобильного приложения

1. Создайте папку `C:\xampp\htdocs\taxi_api`
2. В эту папку перенесите три файла из папки `Файлы XAMPP`: `accept_order.php`, `get_drivers.php`, `get_orders.php`

### 4. Проверка работы PHP скриптов

Откройте браузер и проверьте:

- `http://localhost/taxi_api/get_drivers.php` - должен показать JSON с водителями
- `http://localhost/taxi_api/get_orders.php` - должен показать JSON с новыми заказами

### 5. Установка Python и библиотек

1. Скачайте Python с официального сайта: https://www.python.org/downloads/
2. При установке **обязательно** отметьте галочку **"Add Python to PATH"**
3. После установки откройте командную строку (Win+R → cmd)
4. Установите необходимые библиотеки:

```bash
pip install mysql-connector-python
pip install reportlab
```

### 6. Запуск диспетчерской

1. Распакуйте архив `ПрограммаПК.zip` на компьютере
2. Откройте командную строку и перейдите в папку с файлом:

```bash
cd C:\путь\к\папке
python dispatcher.py
```

3. Откроется окно программы с 4 вкладками:
   - Новый заказ - создание заказов
   - Активные заказы - управление заказами
   - Водители - просмотр списка водителей
   - Выполненные - история и печать отчетов

### 7. Настройка мобильного приложения

### 7.1. Узнайте IP-адрес компьютера

1. На ПК откройте командную строку (Win+R → cmd)
2. Введите команду: `ipconfig`
3. Найдите строку **IPv4-адрес** (например, `192.168.1.35`)

### 7.2. Исходный код проекта

Для работы с мобильным приложением вам понадобится исходный код проекта (архив `ПрограммаМобила.zip`). 

### 7.3. Открытие проекта в Android Studio

1. Распакуйте архив `TaxiDriver_project.zip` в любую папку
2. Запустите Android Studio
3. Нажмите **File → Open**
4. Выберите **распакованную папку** с проектом
5. Нажмите **OK**

### 7.4. Настройка IP-адреса в коде

1. В Android Studio откройте файл `MainActivity.java`
   - Путь: `app/src/main/java/com/example/taxidriver/MainActivity.java`
2. Найдите строку:
   ```java
   private final String BASE_URL = "http://192.168.1.67/taxi_api/";
   ```
   Замените `192.168.1.67` на IP-адрес вашего компьютера

   ```java
   private final String BASE_URL = "http://ВАШ_IP/taxi_api/";
   ```
   Например: `http://192.168.0.105/taxi_api/`

### 7.5. Сборка APK файла

После изменения IP-адреса необходимо собрать новый APK:

1. В Android Studio нажмите **Build → Build Bundle(s) / APK(s) → Build APK(s)**
2. Дождитесь завершения сборки (внизу появится уведомление)
3. Нажмите **Locate** в уведомлении
4. Откроется папка с файлом `app-debug.apk` по пути:
   `TaxiDriver_project/app/build/outputs/apk/debug/`

### 7.6. Установка приложения на телефон

1. Скопируйте файл `app-debug.apk` на телефон
2. На телефоне откройте файловый менеджер и найдите APK файл
3. Нажмите на файл
4. Если система запросит разрешение, разрешите установку из неизвестных источников
5. Нажмите **Установить**

### 7.7. Проверка соединения

Перед запуском приложения проверьте, что API доступен с телефона:

1. Откройте браузер на телефоне
2. Введите адрес:

   ```
   http://[IP_вашего_ПК]/taxi_api/get_orders.php
   ```

   Например: `http://192.168.1.35/taxi_api/get_orders.php`

3. Если вы видите JSON данные (например, `[]` или список заказов) - соединение работает
